Como adicionar meu projeto?

Você tem duas opções:

Fazer uma merge request no Gitlab

1. Vá para o repositorio do projeto
2. Faça um fork do projeto, e baixe na sua máquina
3. Crie uma branch nova para colocar sua página
4. Depois das mudanças faça push da sua branch
5. E então, na UI do Gitlab, vai aparecer a opção para fazer o merge request
